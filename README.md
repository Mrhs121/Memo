# Memo
Django + vue + 前后端分离 备忘录小项目

# 依赖
* Python 3.x
* Django 2.x
* vue 2.x


# 启动

* 修改Todo/Todo/setting.py 文件中数据库的配置
* cd Todo
* python manage.py makemigrations
* python manage.py migrate
* python manage.py runserver

# 注意
本项目只是简单的python小作业，没有写权限认证。
